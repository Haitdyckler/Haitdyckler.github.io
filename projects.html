Arduino Alarm Clock

For My IoT project, I made an Arduino Alarm clock, that ran from an esp32. The circuit was equipped with humidity sensors, 4 buttons to set the time for the alarm, an lcd screen, speaker and a motion sensor.

When equipped with Wi-Fi, I was able to connect the board to the internet and make the clock run on the time based on where it is, and you would simply need a power supply to use the alarm clock, making it very portable. The alarm clock's motion sensor will detect if there is any motion after the alarm has rung, if the user has not moved; the alarm will automatically turn on again.

The reason I chose to do this project is because I've always been interested in circuitry, and this gave me a chance to actually learn and experiment. I took the opportunity to create a functional, reusable, and fun device that utilizes IoT and portability.
